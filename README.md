## Peaks-Over-Threshold, POT 原理

超越門檻模型（Peaks-Over-Threshold, POT）是一種在極端值理論（Extreme Value Theory, EVT）中用以分析罕見事件的重要方法。其核心原理在於，**當觀測數據超過一個足夠高的門檻值時，這些超出門檻的「極端值」（稱為超越量），其分布會趨近於一個特定的統計模型——廣義帕雷托分布（Generalized Pareto Distribution, GPD）**。

簡單來說，POT模型不像傳統方法一樣關注所有數據，而是特別聚焦於那些最極端的觀測值，並利用這些有限的極端數據來推斷未來發生更極端事件的機率。

### 核心理論基礎：Pickands–Balkema–De Haan 定理

POT模型的理論基石是**Pickands–Balkema–De Haan定理**。該定理指出，對於一個隨機變數，只要其門檻值`u`設定得足夠高，那麼變數值`X`超過`u`的量（即`X-u`），其條件機率分布可以被廣義帕雷托分布（GPD）良好地近似。

這一定理的重要性在於，它告訴我們無論原始數據的分布形態為何，其尾部（即極端值）的行為都有一個共通的模式，這個模式可以用只有兩個參數的GPD來描述。

### POT模型的運作步驟

1.  **設定門檻值（Threshold）**：這是整個分析中最關鍵也最具挑戰性的一步。門檻值的選擇需要在偏差（bias）與變異（variance）之間取得平衡。
    *   **門檻太低**：會納入過多非極端的數據點，導致模型產生偏差，不符合GPD的理論假設。
    *   **門檻太高**：雖然數據更符合極端值的定義，但可用的數據點太少，會導致模型估計的誤差增大，結果不穩定。

2.  **識別超越量（Exceedances）**：一旦確定門檻，所有超過該門檻的數據點都被識別出來。分析的重點是這些數據點超出門檻的部分。

3.  **擬合廣義帕雷托分布（GPD）**：利用識別出的超越量數據，來估計GPD模型的兩個關鍵參數：
    *   **尺度參數（Scale Parameter, σ）**：影響超越量分布的廣度。
    *   **形狀參數（Shape Parameter, ξ）**：決定分布尾部的形狀，是判斷極端事件嚴重性的關鍵，它描述了尾部的「厚重」程度。

4.  **風險評估與推斷**：一旦GPD模型建立完成，就可以用來推斷未來可能發生的極端事件，例如計算特定強度的極端事件在未來一段時間內發生的機率（即回歸期）。

### POT模型與其他方法的比較

POT模型常與另一種極端值分析方法——**區間最大值法（Block Maxima Method, BMM）**——進行比較。

*   **數據利用效率**：POT模型更有效率，因為它利用了所有超過門檻的極端數據。相比之下，BMM是將數據分成固定時間區塊（例如每年），然後只取每個區塊中的最大值來分析，這會浪費掉許多次大但仍然很極端的數據點。
*   **門檻選擇**：POT模型的挑戰在於門檻的設定，而BMM則需決定區塊的大小。

### 應用領域

由於POT模型能有效分析罕見但影響重大的事件，因此被廣泛應用於多個領域：

*   **金融與保險**：用來評估市場的極端風險，例如計算風險價值（Value at Risk, VaR），或預測巨額保險理賠的可能性。
*   **水文學與環境科學**：分析極端氣候事件，如預測百年一遇的洪水、極端降雨量或最高風速等。
*   **工程學**：在結構設計中，評估建築物可能承受的極端風力或地震強度。
*   **時序數據異常檢測**：識別系統中罕見但關鍵的異常訊號。
